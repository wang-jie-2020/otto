# 调研指南（Mode A）

本文档提供调研源列表、质量判断标准、以及从调研结果到图解结构的转换方法。

---

## 调研源优先级

### 1. Twitter/X

最新动态和业内大佬直接发声。适合调研技术趋势、产品发布、行业观点。
看粉丝数和认证、转发点赞比、评论区讨论质量。警惕营销号。

### 2. 知乎

深度分析和专业答主集中。适合调研概念解释、技术原理、行业分析。
看答主背景、赞同收藏比、评论区质量。警惕软广和带货。

### 3. YouTube

视频内容直观、专家访谈。适合调研教程、产品评测、会议演讲。
看频道订阅数、播放量、评论质量。警惕标题党。

### 4. 小红书

真实用户体验。适合调研产品体验、生活方式、消费决策。
看粉丝互动、内容真实性。警惕种草文。

### 5. Google / Google Scholar

权威来源、学术文献。适合调研学术概念、官方数据、技术文档。
看来源域名（.gov, .edu）、发布时间、引用次数。警惕假新闻。

## 信息提炼方法

### Step 1: 快速扫描
扫描标题、摘要、作者、时间、互动数。决定是否值得深入。

### Step 2: 提取关键信息

提取框架：
```
来源信息：平台 / 作者 / 标题
核心内容：主要观点 / 关键概念 3-5个 / 流程步骤 / 数据案例
可信度评价
```

### Step 3: 结构化整理

将散乱信息整理成可图解的结构：

| 类型 | 特征 | 适合图表 |
|------|------|---------|
| 流程型 | 有步骤、有顺序 | flowchart |
| 层级型 | 有分类、有上下级 | mindmap |
| 关系型 | 有连接、有交互 | flowchart / sequenceDiagram |
| 对比型 | 有不同方案/选项 | flowchart（并列） |
| 时序型 | 有时间线、有阶段 | journey / sequenceDiagram |

## 调研质量标准

**必须满足**：
- 至少 3 个来源
- 至少 2 个不同平台
- 核心概念有明确定义
- 流程/关系有清晰描述

**加分项**：
- 有权威来源（官方文档、论文）
- 有真实案例或数据
- 有不同观点的对比
- 有生动的比喻或例子

---

## 常见调研陷阱

1. **信息茧房**：只看一个平台/一种观点 → 跨平台搜索，看不同立场
2. **过时信息**：引用几年前的内容 → 检查发布时间，优先最新
3. **软广和营销**：把广告当干货 → 看利益相关，多看几篇对比
4. **知道分子**：作者自己也是半桶水 → 看作者背景，交叉验证
