## 模板变量

| 变量 | 说明 |
|------|------|
| `{选题}` | 话题名称 |
| `{标题}` | 每张图的标题 |
| `{emoji}` | 标题 emoji（L1 为空） |
| `{图表代码}` | 完整 Mermaid 代码 |
| `{date}` | 生成日期 |
| `{模式}` | 调研 / 文本 |
| `{等级}` | 精简/标准/详细/超详细 |
| `{L级别}` | L1/L2/L3/L4 |
| `{简洁}` | 低/中/高 |
| `{配色}` | 暖色系/冷色系/彩虹渐变/黑白极简 |
| `{受众}` | 入门/进阶/专业 |

## 图解数量规则

根据用户选择的数量等级动态调整图表数量：
- 精简：0-6 张
- 标准：6-10 张
- 详细：10-15 张
- 超详细：15+ 张

首页（标题+要点）不计入图表数量。
